# 🎓 Exam Master

Exam Master is a lightweight web application designed to help university students prepare for exams more efficiently.

## ✨ Features

- 📚 Simple and clean user interface
- ⚡ Fast performance powered by Vite
- 🎯 Designed for students and exam preparation
- 🌐 Runs directly in the browser
- 📱 Responsive design

## 🚀 Demo

You can run the project locally:
npm install
npm run dev

Then open:
http://localhost:5173

## 🛠️ Built With

- TypeScript
- Vite
- HTML5
- CSS3

exam-master/

## 📦 Installation

Clone the repository:
git clone https://github.com/amirniki1283/exam-master.git

Navigate into the project:
cd exam-master

Install dependencies:
npm install

Run development server:
npm run dev
